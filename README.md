# HackingBox v2 - Termux Edition 🔥

**Yo, bem-vindo ao HackingBox v2, bradar!**

## Sobre o Projeto
Esse script é um menu em Python pra facilitar a vida de quem curte mexer com ferramentas de pentest e hacking no Termux. Versão 2: mais limpo, com boas-vindas personalizada e opções pra instalar várias tools famosas.

- **Usuário**: Pergunta seu nome no início.
- **Funcionalidades**: Menu com opções pra atualizar Termux, instalar DDoS-Ripper, ZPhisher, NMAP, SQLMap, Metasploit, GhostTrack, Infect, Kali NetHunter, TBomb, CamPhish e mais.

**Aviso importante**: Isso é pra fins educacionais e de aprendizado, se voc^fazer absolutamente qualquer coisa ilegal com ela, VOCê se resposabiliza!

## Como Usar
1. Instala coisas basicas: `pkg install python python-pip - pip install colorama`
2. Salva o código num arquivo tipo: `git clone https://github.com/kzzxp007/HackingBox/`
3. Dá permissão: `chmod +x hackingbox.py`
4. Roda: `python hackingbox.py`

### Menu de Opções:
- **0**: Atualizar Termux
- **1**: DDoS-Ripper
- **2**: ZPhisher
- **3**: NMAP
- **4**: SQLMap
- **5**: Metasploit (instalação completa)
- **6**: GhostTrack
- **7**: Infect (Virus) ⚠️ Cuidado!
- **8**: Kali Linux (NetHunter)
- **9**: TBomb
- **10**: CamPhish
- **00**: Sair

## Requisitos
